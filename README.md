# Election_Analysis

## Project Overview
A Colorado Board of Elections employee has given you the following task to compete the election audit of a recent local congressional election.

1. Calculate the total number of votes cast.
2. Get a complete list of candidates who received votes.
3. Calculate the total number of votes each candidate won.
4. Calculate the percentage of votes each candidate won.
5. Determine the winner of the election based on popular vote.

## Resources
- Data Source: election_results.csv
- Software: Python 3.10.5, Visual Code, 1.38.1

## Summary
The analysis of the eleciton show that:
- The candidates were:
  - Candidate 1
  - Candidate 2
  - Candidate 3
- The candidate results were:
  - Candidate 1 received "x%" of the vote and "y" number of votes.
  - Candidate 2 received "x%" of the vote and "y" number of votes.
  - Candidate 3 received "x%" of the vote and "y" number of votes.
- The winner of the election was:
  - Candidate (1,2, or 3), who received "x%" of the vote and "y" number of votes.
